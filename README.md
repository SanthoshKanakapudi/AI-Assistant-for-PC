AI Assistant for PC

An intelligent voice-controlled AI assistant designed for PC that automates routine tasks, enhances productivity, and enables efficient human–computer interaction.

---

Features

- Voice recognition for real-time command input
- System control to open applications and perform operations
- Web integration for instant search and navigation
- AI-based responses using external APIs
- Text-to-speech output for interactive feedback

---

Tech Stack

- Language: Python
- Libraries: SpeechRecognition, pyttsx3, os, webbrowser
- APIs: AI/Chat APIs (optional integration)

---

How It Works

1. The system captures voice input from the user
2. Speech is converted into text using recognition modules
3. The command is processed through logic or AI models
4. The corresponding operation is executed on the PC
5. The system provides a voice response
